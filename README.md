## New Algebraic Points on Curves

This repository contains the Magma code supporting the paper "New Algebraic Points on Curves" by Maleeha Khawaja and Samir Siksek (https://arxiv.org/abs/2511.15635).

* The file hyp.m contains the code supporting the proof of Corollary 5.
* The file X023.m contains the code supporting the proof of Theorem 8 for X_0(23).
* The file X029.m contains the code supporting the proof of Theorem 8 for X_0(29).
* The file X031.m contains the code supporting the proof of Theorem 8 for X_0(31).
* The file X064.m contains the code supporting the proof of Theorem 8 for X_0(64).
