## New Algebraic Points on Curves

This repository contains the Magma code supporting the paper "New Algebraic Points on Curves" by Maleeha Khawaja and Samir Siksek.

* The file X023.m contains the code supporting the proof of Theorem 5 for X_0(23).
* The file X029.m contains the code supporting the proof of Theorem 5 for X_0(29).
* The file X031.m contains the code supporting the proof of Theorem 5 for X_0(31).
